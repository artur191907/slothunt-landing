# SlotHunt Landing

Лендинг сервиса автоматического отлова слотов перераспределения остатков на Wildberries.

## Технологии

- Next.js 14
- React 18
- Tailwind CSS
- Recharts (графики)
- Lucide React (иконки)

## Локальная разработка

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Деплой

Автоматически разворачивается на Vercel при push в main.
